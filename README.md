# Prerequisites

Ensure you have the following installed on your system:

Node.js: v16 or later. npm or yarn: Latest stable version.

## Available Scripts

In the project directory, you can run:

### `git clone https://github.com/baranozbekk/movie-app.git`

### `cd movie-app`

### `npm install`

or

### `yarn install`

### `npm start`

or

### `yarn start`
